# Asset Checklist for Chrono Squad

## High Priority
1. **Character Sprites**  
   - Main Character Sprite  
   - Secondary Characters Sprites  

2. **Animations**  
   - Idle Animations  
   - Walk Animations  
   - Attack Animations  
   - Special Move Animations  

3. **Enemy Sprites**  
   - Basic Enemy Sprites  
   - Boss Enemy Sprites  

## Medium Priority
4. **UI Elements**  
   - Menus  
   - Health Bar  
   - Score Display  
   - Inventory UI  

5. **Backgrounds**  
   - Level 1 Background  
   - Level 2 Background  
   - Parallax Layers  

## Low Priority
6. **Sound Effects**  
   - Jump Sound Effect  
   - Attack Sound Effect  
   - Collectible Sound Effect  

7. **Music Tracks**  
   - Level 1 Background Music  
   - Level 2 Background Music  
   - Boss Fight Music  

## Creation Order  
1. Character Sprites  
2. Animations  
3. Enemy Sprites  
4. UI Elements  
5. Backgrounds  
6. Sound Effects  
7. Music Tracks  

---  
This checklist is to ensure that all necessary assets are created in a structured and organized manner to streamline the development process.
